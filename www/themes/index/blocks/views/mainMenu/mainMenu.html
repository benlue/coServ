Index Page

This page is served by a static block. That means contents are neither coming from a local module nor from a API service call.

The HTML source can be found in blocks/views/index/index.html

Cross Origin

In this example, contents are served by a local module (blocks/modules/co/rss.js). That module loads a RSS feed from Yahoo, parses it and then sends it out as a JSON list for the template engine to compose the HTML output as you see it.

The source file of the local module can be found in blocks/modules/co/rss.js, and HTML source can be found in blocks/views/co/rss/rss.htlml. There is also a rss.js file in the 'blocks/views/co/rss' directory. That js file will run on browsers. Do not confuse it with the rss.js node module.

Block Reusing

This example shows how to re-use a block. The RSS block used in the 'Cross Origin' example above is used again in this example, not once but TWICE. Please go checking the /blocks/views/resue/reuse.html file to see how easily and elegantly this can be done. You may even customize its style when reusing a block as shown in this example.
