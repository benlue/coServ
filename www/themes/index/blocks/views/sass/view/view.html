coServ v0.8.8 新功能

內建SASS

支援scss語法，不用自行編譯或下額外指令

關於SASS

官方說明

簡單來說，讓CSS寫法更具彈性

語法

長相？

@mixin border-radius($radius) {
  • -webkit-border-radius: $radius;
  • -moz-border-radius: $radius;
  • -ms-border-radius: $radius;
  • -o-border-radius: $radius;
  • border-radius: $radius;
} button {
  • @include border-radius(5px);
}

如何使用

在coServ，SCSS有兩種用法

  • 部分使用
    • blocks
  • 共用
    • cont/shared/sass

部分使用

路徑：blocks/views/[BlockName]

網頁切板常以區塊區分，這也是coServ載入頁面的最小單位，而一個block 會有這些檔案
  • Blocks
    • myBlock
      • myBlock.html
      • myBlock.css
      • myBlock.js
      • myBlock.lang /* 多國語言設定檔 */
      • myBlock.sass /* Here! */

共用

路徑：cont/shared/sass

實際上是cont/shared/sass，適合放mixin library或自定的mixin.scss，讓myBlock.sass能夠引用sass code

如何運作

載入順序

你可能會有疑問，有CSS又有SCSS，那coServ會怎麼處理呢 ？ 先載入Blocks內的.css，沒有才載入. 但coServ直接幫你在網頁加上sass編譯完成的css code，而不用另存新檔。

擴充SASS MIXIN

Mixin Library

Compass
Bourbon

來喝酒吧

  1. npm install node-bourbon --save
  2. find "drink bourbon", 移除註解
  3. /* myBlock.scss */
    4. @import "bourbon"; /* 載入bourbon mixin*/
    • ....
    • .
    • .
    • ....[your scss code]

這樣就能使用Bourbon的mixin了！
